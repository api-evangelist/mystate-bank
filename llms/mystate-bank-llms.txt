# MyState Bank

> MyState Bank is an Australian retail bank headquartered in Hobart, Tasmania, and the principal banking brand of ASX-listed MyState Limited (ASX:MYS). As a regulated authorised deposit-taking institution (ADI) and Consumer Data Right (CDR) data holder, it exposes a public, unauthenticated Product Reference Data (PRD) API conforming to the Consumer Data Standards (CDS). The PRD API is read-only and requires no authentication; consumer data sharing beyond PRD is governed by the CDR accredited-data-recipient model.

## APIs

- [MyState Bank CDR Product Reference Data API](https://consumerdatastandardsaustralia.github.io/standards/#banking-products): Get Products and Get Product Detail. Base URL https://public.cdr.mystate.com.au/cds-au/v1 (confirmed live, HTTP 200, x-v 3). Unauthenticated, read-only.
